# Обновление Gradle

## Проблема
Использовалась несовместимая версия Gradle 8.2 с Java 21.

## Решение
Обновлен Gradle до версии 8.5, которая поддерживает Java 21.

## Изменения
- `gradle/wrapper/gradle-wrapper.properties`: обновлена версия с 8.2 до 8.5

## Следующие шаги
1. В Android Studio нажмите **Sync Now** или **File → Sync Project with Gradle Files**
2. Gradle автоматически скачает версию 8.5 при первой синхронизации
3. После синхронизации проект должен собраться без ошибок

## Совместимость версий
- ✅ Gradle 8.5 поддерживает Java 17-21
- ✅ Android Gradle Plugin 8.2.0 совместим с Gradle 8.5
- ✅ Все зависимости остаются без изменений
